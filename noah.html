N.O.A.H is a team management device that allows users to assign tasks to team members, track progress of tasks, and signal other members if help is needed.

Using an Arduino device and by establishing a wireless connection, my team and I created the first N.O.A.H prototype.

The goal of the device is to help teams better manage group work by making sure everyone is aware of each member's current status, making sure each member has a task to work on so that goals can be reached efficiently, and alerting members when help is needed so that tasks can be completed in a timely fashion.
